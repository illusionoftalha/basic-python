# Basic Python

> A two-tier Python learning resource — standalone `.py` files covering everything from beginner fundamentals to advanced concepts.

---

## Overview

A structured collection of Python scripts organized into two folders by difficulty level. Each file focuses on a single concept, making it easy to jump to exactly what you need. Whether you're writing your first `print()` or exploring decorators and file I/O, this repo has it covered.

---

## Structure

```
basic-python/
├── Python for Beginners/   # Core fundamentals — start here
└── Python Advanced/        # Intermediate to advanced concepts
```

---

## Topics Covered

### Python for Beginners
- Variables and data types (`int`, `float`, `str`, `bool`)
- User input and output
- Operators (arithmetic, comparison, logical)
- Conditional statements (`if`, `elif`, `else`)
- Loops (`for`, `while`, `break`, `continue`)
- Functions (defining, calling, parameters, return values)
- Lists, tuples, sets, and dictionaries
- String methods and formatting

### Python Advanced
- List comprehensions
- Lambda functions
- `*args` and `**kwargs`
- Exception handling (`try`, `except`, `finally`)
- File I/O (reading and writing files)
- Modules and imports
- Object-Oriented Programming (classes, objects, inheritance)
- Iterators and generators

## Getting Started

### Prerequisites

- Python 3.x installed — [download here](https://www.python.org/downloads/)

### Run any file

```bash
cd "Python for Beginners"
python filename.py
```
